# Brutalist & Anti-Design

*8 styles in this family (2 flagship). Generated from `scripts/style_catalog.json` — the single source of truth.*

Styles that reject polish on purpose: raw HTML honesty, thick borders, hard offset shadows, clashing color, default system type. They range from the austere (web brutalism) to the playful (neubrutalism) to the confrontational (anti-design, punk). The 'rules' being broken are real rules — you must know them to break them coherently.

> **How to apply this family.** **Consistency is what separates 'brutalist' from 'broken'.** Even chaos needs a system: pick your border weight, shadow offset, and palette and repeat them. Keep real focus outlines and link underlines — anti-design is not an excuse to fail accessibility.

### Brutalist Web Design  ⭐ *flagship — full spec in [00-flagship-implementation-specs/brutalism.md](00-flagship-implementation-specs/brutalism.md)*
*Aliases:* web brutalism, raw web  
*Slug:* `brutalism` · *Era:* 2014–present

**Origin.** Named after architectural Brutalism (béton brut); web movement popularized via brutalistwebsites.com (Pascal Deville).

**Defining traits.**
- Raw, unstyled or near-default HTML aesthetics
- System fonts, default blue links, visible structure
- High contrast, monospace, hard edges, no rounding
- Deliberately 'undesigned', utilitarian honesty

**Reference example.** brutalistwebsites.com; Bloomberg Businessweek features; Craigslist ethos.

**Don't confuse with:** neubrutalism, minimalism.

### Neubrutalism  ⭐ *flagship — full spec in [00-flagship-implementation-specs/neubrutalism.md](00-flagship-implementation-specs/neubrutalism.md)*
*Aliases:* neo-brutalism, new brutalism UI  
*Slug:* `neubrutalism` · *Era:* 2020–present

**Origin.** Rose with Gumroad's 2021 redesign; heavily used by Figma community.

**Defining traits.**
- Thick solid black borders and hard offset drop shadows (no blur)
- Loud, clashing saturated colors and flat fills
- Zero-to-chunky border-radius, bold grotesk type
- Playful, intentionally 'ugly-cute', high energy

**Reference example.** Gumroad (2021 redesign); Figma marketing; many indie SaaS.

**Don't confuse with:** brutalism, claymorphism, memphis-design.

### Anti-Design
*Aliases:* ugly design, undesign  
*Slug:* `anti-design` · *Era:* 2016–present

**Origin.** Reaction against polished 'design-by-committee' minimalism.

**Defining traits.**
- Deliberate rule-breaking: clashing type, misalignment
- Default/system styling embraced
- Rejects visual hierarchy conventions
- Provocative, expressive, DIY

**Reference example.** Balenciaga site experiments; art/fashion microsites.

**Don't confuse with:** brutalism, dadaist-web.

### Dadaist / Collage Web
*Aliases:* dada web, collage UI  
*Slug:* `dadaist-web` · *Era:* 2015–present (from 1916 Dada)

**Origin.** Digital revival of Dada photomontage and chance-based layout.

**Defining traits.**
- Cut-and-paste photomontage, ransom-note type
- Absurd juxtaposition and chaos
- Mixed media scraps and stickers
- Anti-rational composition

**Reference example.** Art-school microsites; fashion editorial pages.

**Don't confuse with:** anti-design, punk-zine.

### Deconstructivism (Graphic)
*Aliases:* deconstruction, fragmented type  
*Slug:* `deconstructivism` · *Era:* 1980s–1990s

**Origin.** Cranbrook Academy; influenced by Derrida; Katherine McCoy, Edward Fella.

**Defining traits.**
- Fragmented, layered, exploded typography
- Rejection of the grid; ambiguous reading paths
- Overlap, rotation, and collision of elements
- Theory-driven, questions legibility

**Reference example.** Cranbrook design work; Emigre magazine.

**Don't confuse with:** swiss-punk, grunge.

### Grunge Design
*Aliases:* dirty design, distressed design  
*Slug:* `grunge` · *Era:* 1990s

**Origin.** David Carson (Ray Gun magazine); reaction to clean corporate design.

**Defining traits.**
- Distressed textures, dirt, splatter, torn edges
- Chaotic overlapping type, broken legibility
- Photocopied, degraded, layered collage
- Anti-grid experimental layout

**Reference example.** David Carson, Ray Gun magazine (1990s).

**Don't confuse with:** swiss-punk, glitch-art.

### Raw / Default HTML
*Aliases:* unstyled web, default styles, HTML brutalism  
*Slug:* `raw-html` · *Era:* 1991–present (revival 2016+)

**Origin.** The literal default browser stylesheet; revived as an aesthetic statement.

**Defining traits.**
- Times New Roman, default blue/purple links
- No CSS or minimal CSS
- Single-column, top-to-bottom flow
- Radical performance and honesty

**Reference example.** motherfuckingwebsite.com; bettermotherfuckingwebsite.com.

**Don't confuse with:** brutalism, minimalism.

### Swiss Punk / New Wave
*Aliases:* new wave typography, punk typography  
*Slug:* `swiss-punk` · *Era:* 1978–1990

**Origin.** Wolfgang Weingart (Basel), spread by April Greiman and Dan Friedman in the US.

**Defining traits.**
- Deconstructed Swiss grid; tilted, layered type
- Wide letter-spacing, stepped baselines, rules
- Photographic collage, gradients (Greiman's digital work)
- Rebellion against rigid International Style

**Reference example.** April Greiman 'Does It Make Sense?' (1986); Weingart Typographische Monatsblätter.

**Don't confuse with:** swiss-design, deconstructivism.
